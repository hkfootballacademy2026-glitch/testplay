📱 シリーズ共通の設置方法と安全
  1. スマホを倒れないよう正面に固定し、2.5〜3m離れて全身を映す。
  2. 人物の後ろにバーチャルステージとゴールが合成されます。キャリブレーションで固定された仮想ボールを、実際のボールと同じように自然なフォームで空振りします。
  3. 踏み込みは1歩以内。実物のボールは置かず、スマホや家具へ近づき過ぎない。
  4. 周囲2m以上を空け、家具・人・ペットがいないことを確認する。横画面推奨。
  5. インカメラは画面を確認しながらプレー可能。アウトカメラは補助者がいる時に推奨。
🦵 ボールが変化する仕組み
  • 振り抜き方向＋軸足の向き＋当たった左右位置 → 初速ベクトルと横回転。
  • ボール下部／上部へのインパクト＋足先の上下動 → 打ち出し角、バックスピン、トップスピン。
  • インサイド／インフロント／アウトフロント／アウトサイド／インステップ／トー → 衝突効率と3軸回転。
  • スイング速度、奥行き方向、膝の伸展、軸足、足首の固定、姿勢 → ボール初速、打ち出し方向、再現性。
  • 発射後は重力・空気抵抗・マグヌス効果・地面反発・ポスト衝突を時間積分し、カーブ、チップ、ドライブ、ナックルなどの球質として画面へ投影します。
